iba-x-demo

Run X's real recommendation algorithm safely under IBA governance

Combines xAI's open recommendation engine (Phoenix + Grox) with Intent-Bound Authorization (IBA) — so every major action requires your signed human intent.

Quick Start

git clone https://github.com/Grokipaedia/iba-x-demo.git
cd iba-x-demo

git clone https://github.com/xai-org/x-algorithm.git x-algorithm

cd x-algorithm/phoenix
uv run run_pipeline.py --help
